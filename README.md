## Install

- Fork project
- Clone your fork:

```bash
git clone <link-to-project>
cd <project-directory>/
npm install
```

## Environment

Create a .env file with:

- REACT_APP_AUTH_API [go here](https://github.com/weyvern/auth-api-wd012)

# Commands

## Dev mode

npm start
